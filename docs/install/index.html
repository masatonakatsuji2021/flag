Installation

There are two ways to install it: installing from the npm command and downloading the source directly from GitHub.
Installation from the npm command is recommended.

- Install from npm command

First make sure Node.js (npm) is pre-installed.
Supported Node.js versions are Node.js 14 and newer

If Node.js is not installed, please install hereNode.js.

All you have to do is open the console and install the flag body with the following npm command

npm i -g @flagfw/flag

The installation is complete when all the necessary npm packages are automatically installed during installation.
